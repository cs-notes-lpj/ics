
ELF: Executable and Linkable Format

程序工具 `xxd` 用于将二进制文件以十六进制形式进行输出

```bash
tldr xxd
Create a hexadecimal representation (hexdump) from a binary file, or vice-versa.
...
```

程序工具 `file` 可用于查看文件类型

```bash
tldr file
Determine file type.
...
```

程序工具 `objdump` 可用于反汇编二进制文件

```bash
tldr objdump
View information about object files.
...
```
